# Money Manager - JavaFX

A modern, multi-user personal finance tracker built with JavaFX.

## Features
- User authentication (login/signup)
- Multi-account support (Wallet, Bank, Mobile Banking)
- Income, Expense, and Transfer transactions
- Statistics and charts (Pie, Bar, Line charts)
- JSON-based data persistence
- Beautiful, modern UI with Ikonli icons

## Project Structure
- `src/models/` - Data models (User, Account, Transaction, etc.)
- `src/services/` - Business logic (Auth, Ledger, File I/O)
- `src/controllers/` - JavaFX controllers
- `resources/fxml/` - FXML view files
- `resources/css/` - Styling
- `resources/data/` - JSON data files

## Setup & Run
1. Clone the repo
2. Open in IntelliJ IDEA
3. Run `Main.java`

## Tech Stack
- Java 11+
- JavaFX 21
- Ikonli (for icons)
- JSON (via Gson or Jackson)
